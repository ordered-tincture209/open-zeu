# Project Template

Create a new project:

```bash
./scripts/new-project.sh <slug> "<Display Name>" "<PM Name>" "<Description>"
```

Example:

```bash
./scripts/new-project.sh pm5 "PM5" "PM5" "Customer work"
```

After that:

1. Fill the placeholders in `projects/<slug>/AGENTS.md`.
2. Add the project to `ui/server.py`.
3. Add your first task and RD.
